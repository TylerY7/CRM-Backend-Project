Available Endpoints

  • GET /customers/{id}

    Retrieves a single customer in the mock database by the customer's ID provided by the user

  • GET /customers

    Retrieves all customers

  • CREATE /customers

    Users provides customer attributes and creates a new customer to the mock database

  • POST /customers/{id}

    Updates am existing customer by their ID

  • DELETE /customers/{id}

    Removes a specific customer by their ID
